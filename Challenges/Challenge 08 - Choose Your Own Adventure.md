# Challenge 08 - Choose Your Own Adventure
The object of this challenge is to create a short choose your own adventure story. Believe it or not, this is what video games where actually like back in the day!

The general strucure of this challenge will be providing the user with a scenario and 2 choices. THe user inputs a choice, and the computer presents the next scenario. Each level of the challenge just adds one more level of depth.

## Mild 🌶️
Create a program that satisfies the following requirements:

* The user makes a total of 2 choices throughout the game, reulting in 4 possible endings.
* The program prevents the user from typing invalid values and handles errors appropriately.
* The program only moves forward if input is valid.

## Medium 🌶️🌶️
Create a program that satisfies the following requirements:

* The user makes a total of 3 choices throughout the game, resulting in 8 possible endings.
* The program prevents the user from typing invalid values and handles errors appropriately.
* The program only moves forward if input is valid.

## Spicy 🌶️🌶️🌶️
Create a program that satisfies the following requirements:

* The user makes a total of 4 choices throughout the game, resulting in 16 possible choices.
* The program prevents the user from typing invalid values and handles errors appropriately.
* The program only moves forward if input is valid.
