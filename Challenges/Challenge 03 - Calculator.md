# Challenge 03 - Calculator
The object of this challenge is to create a simple calculator where the user can input desired numbers and operations.

## Mild 🌶️
Create a program that satisfies the following requirements:

* The program prompts the user to input 2 values.
* The program prompts the user to input an operation (`+`, `-`, `*`, `/`).
* The program performs the calculation and prints the result.
* The program prevents the user from typing invalid values and handles errors appropriately.
* The program only moves forward if input is valid.

## Medium 🌶️🌶️
Create a program that satisfies the following requirements:

* The program prompts the user to input 2 values.
* The program prompts the user to input an operation (`+`, `-`, `*`, `/`).
* The program performs the calculation and prints the result.
* The program asks the user if they would like to perform another calculation and repeats the process if they do.
* The program prevents the user from typing invalid values and handles errors appropriately.
* The program only moves forward if input is valid.

## Spicy 🌶️🌶️🌶️
Create a program that satisfies the following requirements:

* The program prompts the user to input 2 values.
* The program prompts the user to input an operation (`+`, `-`, `*`, `/`, `%`, `//`, `**`).
* The program performs the calculation and prints the result.
* The program asks the user if they would like to perform another calculation and repeats the process if they do.
* The program prevents the user from typing invalid values and handles errors appropriately.
* The program only moves forward if input is valid.
