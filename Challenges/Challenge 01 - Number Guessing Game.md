# Challenge 01 - Number Guessing Game
The object of this challenge is to create a game where there is a predetermined number and the user inputs guesses.

## Mild 🌶️
Create a program that satisfies the following requirements:

* The program chooses a random number with `random.randint()` between 1 and 5 *(you may have to research this)*.
* The user can input a guess through the terminal.
* If the user is correct, the program prints a "Win" message.
* If the user is incorrect, the program prints a "Lose" message.
* The program prevents the user from typing invalid values and handles errors appropriately.
* The program only moves forward if input is valid.

## Medium 🌶️🌶️
Create a program that satisfies the following requirements:

* The program chooses a random number with `random.randint()` between 1 and 100 *(you may have to research this)*.
* The user inputs a guess through the terminal.
* If the user is correct, the program prints a "Win" message.
* If the user is incorrect, the program prints a message saying if the guess was too high or too low through a printed message to the terminal.
* The user is given the opportunity to continue guessing until they guess the correct number.
* The program prevents the user from typing invalid values and handles errors appropriately.
* The program only moves forward if input is valid.


## Spicy 🌶️🌶️🌶️
Create a program that satisfies the following requirements:

* The program chooses a random number with `random.randint()` between 1 and 100 *(you may have to research this)*.
* The user inputs a guess through the terminal.
* If the user is correct, the program prints a "Win" message. The win message also tells the user how many guesses it took to get the correct answer.
* If the user is incorrect, the program prints a message saying if the guess was too high or too low through a printed message to the terminal.
* The user is given the opportunity to continue guessing until they guess the correct number.
* When the user has guessed the correct number, the program prompts them about whether they want to play again. If they want to play again, the program chooses a new number and restarts the process.
* The program prevents the user from typing invalid values and handles errors appropriately.
* The program only moves forward if input is valid.
