# Challenge 04 - Coin Flip Game
The object of this challenge is to create a game where the user guesses the result of a coin flip.

## Mild 🌶️
Create a program that satisfies the following requirements:

* The program secretly randomly selects between `"heads"` and `"tails"`.
* The user guesses "heads" or "tails".
* The program prints a message about whether the player was correct or not.
* The program prevents the user from typing invalid values and handles errors appropriately.
* The program only moves forward if input is valid.

## Medium 🌶️🌶️
Create a program that satisfies the following requirements:

* The program secretly randomly selects between `"heads"` and `"tails"`.
* The user guesses "heads" or "tails".
* The program prints a message about whether the player was correct or not.
* The game repeats 10 times.
* After the tenth game, the program prints how many guesses the user got correct out of 10.
* The program prevents the user from typing invalid values and handles errors appropriately.
* The program only moves forward if input is valid.

## Spicy 🌶️🌶️🌶️
Create a program that satisfies the following requirements:

* The program secretly randomly selects between `"heads"` and `"tails"`.
* The user guesses "heads" or "tails".
* The program prints a message about whether the player was correct or not.
* The game continues until the user has guessed incorrectly 3 times in a row.
* The game keeps track of a score. Every correct guess gives the user 1 points. Every incorrect guess gives the user -1 points.
* If the user gets 5 guesses correct in a row, their score gets doubled. Then the count of correct guesses in a row is reset.
* The program prevents the user from typing invalid values and handles errors appropriately.
* The program only moves forward if input is valid.
