# Challenge 07 - Password Strength
The object of this challenge is to test a user's password strength.

## Mild 🌶️
Create a program that satisfies the following requirements:

* The user inputs a proposed password.
* The program verifies that the proposed password has at least the following:
    * 8 characters long
    * 1 uppercase letter
    * 1 lowercase letter
* If the password is missing any of the criteria, the program prints what they are missing and lets them try again.
* The program prevents the user from typing invalid values and handles errors appropriately.
* The program only moves forward if input is valid.

    * 1 lowercase letter

    * does not contain the word "password" within the password
